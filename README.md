# 💧 Water Sort Puzzle

Free, no-signup, language-independent color sorting puzzle. Play instantly in your browser.

**Live demo:** _add your Vercel URL here_

## Features

- 🎮 Endless procedurally generated levels
- 💾 Progress saved to localStorage (no DB, no account)
- ↶ Undo / Reset
- 📱 Works on mobile and desktop
- 🌍 No language barrier — pure visual gameplay

## Stack

- Next.js 15 (App Router) + React 19 + TypeScript
- 100% client-side, deploys to Vercel with zero config

## Local dev

```bash
npm install
npm run dev
# open http://localhost:3000
```

## Deploy

Push to GitHub → import the repo at https://vercel.com/new → done. No env vars required.
